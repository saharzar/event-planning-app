# Evently - Event Management Application

## Description

Evently is a modern event management web application built with Angular and Spring Boot.
Users can create, manage, publish, browse, and join events through a clean and responsive interface.

---

## Technologies Used

### Frontend

* Angular 21
* Bootstrap
* TypeScript

### Backend

* Java
* Spring Boot
* Spring Data JPA
* Swagger
* DTO Architecture
* Validation

### Database

* H2 Database

---

## Features

* User Authentication
* Create & Manage Events
* Browse & Join Events
* Event Status Management
* Search & Filter Events
* Responsive Dashboard
* Session Authentication
